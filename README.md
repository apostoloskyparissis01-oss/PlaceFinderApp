# PlaceFinderApp

Εφαρμογή αναζήτησης μαγαζιών και εστιατορίων βάσει budget, 
τύπου κουζίνας και απόστασης χρησιμοποιώντας το Google Places API.

## Τεχνολογίες

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Python, Flask
- **API:** Google Places API

## Λειτουργίες

- Αναζήτηση μαγαζιών βάσει τοποθεσίας
- Φιλτράρισμα με budget
- Φιλτράρισμα με τύπο κουζίνας
- Φιλτράρισμα με απόσταση
- Προβολή κριτικών και ωραρίου

## Εγκατάσταση

1. Clone το repository
git clone https://github.com/apostoloskyparissis01-oss/PlaceFinderApp.git

2. Εγκατάσταση dependencies
pip install -r requirements.txt

3. Δημιούργησε .env αρχείο στον φάκελο backend/
GOOGLE_API_KEY=το_κλειδί_σου

4. Τρέξε τον server
cd backend
python app.py
